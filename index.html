JQuery Rescue

by Mike Simmonds

Rescue is a JQuery plugin that allows a user to autosave and recover the contents of a form, on the client-side, using HTML5 local storage. View on GitHub or download.

You can also clone the project with Git by running:

$ git clone git://github.com/MikeSimmonds/JQuery-Rescue

Take a look at the README file for setup and configuration options.

Demo

Edit the form below and then refresh the page. Normally data would be lost, but with Rescue the data is recovered and the form restored to it's previous state.

Tip: Take a look at your browser's console.
