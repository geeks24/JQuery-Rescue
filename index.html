JQuery-Rescue by MikeSimmonds

Rescue is a JQuery plugin that allows a user to autosave and recover the contents of a form, on the client-side, using HTML5 local storage.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/MikeSimmonds/JQuery-Rescue
